# Codebase Map

- `src/core/`: autonomous agent core and provider config.
- `src/tools/`: execution, file, and browser action tools.
- `src/memory/`: local persistence and session state.
- `src/health/`: runtime health and resilience.
- `src/ui/server.js`: WebUI API endpoints and websocket updates.
- `tests/`: smoke and WebUI chat e2e checks.
- `deploy/` + `scripts/`: systemd packaging and operational helpers.
